# Daikibo Factory Downtime Analysis

## Project Overview
This project analyzes machine telemetry data collected from multiple Daikibo factories. The goal was to identify which factory experienced the highest machine downtime and which machine types contributed most to these failures.

## Business Problem
Daikibo collected telemetry data from machines across four factories. The management wanted to understand where machines were breaking down most frequently in order to improve operational efficiency.

## Objectives
- Identify the factory with the highest downtime
- Determine which machine types contributed the most to downtime
- Present insights through a clear visual dashboard

## Tools Used
- Tableau
- Data Visualization
- Data Aggregation

## Dashboard
The dashboard displays:
- Down Time per Factory
- Down Time per Device Type

These visuals help quickly identify which factory and machine types require operational improvements.
